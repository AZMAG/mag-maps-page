Land Use Explorer Map Viewer

The Land Use Map Viewer is one of the Maricopa Association of Governments interactive mapping and analysis sites, showing land use data. The viewer allows you to graphically view land use and development data for all or virtually any contiguous subset of Maricopa County and Pinal County, Arizona.

The viewer provides interactive mapping and reporting tools to allow exploration of the MAG 2017 Existing Land Use, Future Land Use, and Developments datasets. The Existing Land Use dataset is created by integrating GIS datasets from a variety of sources including the Maricopa County Assessor's Office, Arizona State Trust and Federal ownership. The Future Land Use dataset is created by overriding areas with developable land uses (e.g. vacant, agriculture) with anticipated land uses prescribed within municipal developments and general plans. The Developments dataset is created by compiling proposed development projects from MAG Member Agencies
